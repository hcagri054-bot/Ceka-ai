# Çeka 🤖

Çeka, dünyanın en güçlü yapay zeka modellerini tek bir çatı altında toplayan ve bir API aracılığıyla çalışan gelişmiş bir yapay zeka asistanı uygulamasıdır.



## 🌟 Desteklenen Yapay Zeka Modelleri

Uygulama, API entegrasyonu sayesinde aşağıdaki lider yapay zeka servisleriyle kesintisiz bir şekilde haberleşir:

* **ChatGPT** (OpenAI)
* **Gemini** (Google)
* **Claude** (Anthropic)
* **DeepSeek**
* **Groq** (Yüksek hızlı çıkarım motoru)
* **Grok** (xAI)

---

## 📱 Kurulum ve Kullanım

### 1. Android (.apk) Kurulumu
Uygulamanın Android sürümünü cihazınıza yüklemek için:
1.  Bu depodaki **`Çeka.apk`** dosyasını indirin.
2.  Cihazınızda "Bilinmeyen Kaynaklardan Yükleme" izninin açık olduğundan emin olun.
3.  Dosyayı açarak yükleme işlemini tamamlayın.

### 2. Web (.html) Sürümü
Uygulamayı tarayıcı üzerinden çalıştırmak için:
1.  Depodaki **`Çeka.html`** dosyasını bilgisayarınıza indirin.
2.  Dosyaya çift tıklayarak herhangi bir modern tarayıcıda (Chrome, Edge, Safari, Firefox) doğrudan çalıştırın.

---

## 🛠️ Nasıl Çalışır?

Çeka, kullanıcıdan aldığı komutları veya soruları arka planda entegre edilmiş olan API sistemine iletir. Seçtiğiniz modele göre (örneğin ChatGPT veya DeepSeek) ilgili yapay zeka motoru sorguyu işler ve sonucu saniyeler içinde kullanıcı arayüzüne geri döndürür.

> **Not:** Uygulamayı tam performansıyla kullanabilmek için ilgili servislerin API anahtarlarına (API Keys) ihtiyacınız olabilir.
